WEB DEVELOPMENT TRAINING PROJECT

Steps to run the project are:
  1. Download any IDE of your choice and import the folder ‘ctrlbudget’ and all the files in it.
  2. Download XAMPP/WAMP server and start apache and MySQL and open phpMyAdmin then import the file named as ‘expense.sql’ as new database. Also the project in htdocs folder of particular server.
  3. In your IDE run the above project that you have imported and website will run successfully.
Note: Do not stop apache and MySQL while you are executing the project and keep connected to internet otherwise the bootstrap components i.e user interface will not be displayed properly.
